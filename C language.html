Variables : Variable is a name of a memory loaction that stores some data

Rules for writing a variable :

  • Variables are case sensitive
  • 1st character is always an alphabet or "_"
  • there should be no comma/blank space
  • there should be no other symbol other than "_" .

Keywords: These are words that have special meaning inside the compiler.There are a total of 32 keywords in c.

Hello world program in c
#include<stdio.h>
int main(){
printf("Hello World") ;
return 0; }

In the above given code <stdio.h> is known as preprocessor directive. Without this line our program won't work.

example 2:
#include<stdio.h>
int main(){
int a =2;
float pi = 3.14;
char ch = 't';
printf("value of a is %d" , a) ;
printf("value of pi is %f" , pi);
printf("value of ch is %c" , ch);
return 0; }

In c language %d , %f , %c are known as format specifiers and are used to print int ,float , char variable respectively. For more format specifiers click here
